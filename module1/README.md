# Домашнее задание №1

## Условие

Написать консольное приложение, которое будет первым параметром принимать 
имя пользователя, вторым - способ получения прогноза и печатать для него 
прогноз. Варианты способа получения **Basic** и **Secure**.

В приложении должно храниться два возможных варианта
**"У вас сегодня будет удача в делах!"**
и **"Сегодня хороший день для саморазвития!"**

Варианты можно хранить в любой удобной для вас структуре данных. 
Если выбран вариант **Basic**, то нужно использовать класс **Random** для получения прогноза, 
а если **Secure**, то класс **SecureRandom**.

## Запуск

```bash
./gradlew clean build
java -jar build/libs/module1-1.0.0-SNAPSHOT.jar Евгений Secure
```
